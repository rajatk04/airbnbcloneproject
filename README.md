# AIRBNBCLONEPROJECT

# 1.0 Overview
# The AirbnbCloneProject is a full-stack web application that replicates the core functionalities of the popular Airbnb platform. This project is built using the MERN stack (MongoDB, Express.js, Node.js) and aims to provide a seamless user experience for booking and listing rental properties.
